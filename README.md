# GitDex — GitHub Profile Finder

A React app that fetches real-time GitHub user profiles using GitHub's public REST API.

## Live Demo
https://git-dex-gd.netlify.app

## Tech Stack
React JS | JavaScript | GitHub REST API | CSS

## Features
- Search any GitHub username
- View profile — avatar, bio, location, followers, repos
- Top 5 repositories with stars and language
- Error handling for invalid usernames
- Fully responsive design
